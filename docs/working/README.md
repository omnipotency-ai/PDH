# Working Directory

This folder holds **temporary files** for active sessions.

Files here are deleted when their purpose is complete. Do not store permanent documentation here.

The `templates/` subfolder contains permanent templates. Copy them into this directory when starting a new task — do not edit the templates directly.

## Templates

- `templates/prd-template.md` — Product requirements document scaffold
- `templates/implementation-plan-template.md` — Task plan with verification steps
- `templates/session-handover-template.md` — End-of-session handover notes
- `templates/working-memory-template.md` — In-session scratchpad
